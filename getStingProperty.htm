getStringProperty

Fonction

Retourne une chaîne unique contenant le nom du pilote de l'équipement ou l'adresse de départ de la table de dialogue.

Format

string EquipmentName.getStringProperty(String param);

Paramètre

Paramètre

Description

String param

Une variable de chaîne qui représente les informations de configuration demandées par l'équipement.

  • Con_Protocol

  • Con_DialogTable_StartAddress

Exemple :

ModbusEquipment01.getStringProperty("Con_Protocol");

Valeur retournée

Retourne une chaîne contenant l'information sur l'équipement. Ou, retourne une chaîne vide.

Voir également

commitConfig, getProperty, openConfig, setProperty
