18.9.2 Méthodes Java

18.9.2.1 Méthodes Math

Vijeo-Designer prend en charge les méthodes Math suivantes à partir de la classe mathématique standard.

Pour utiliser ces méthodes dans un script Vijeo-Designer, nommez l'ensemble Math, la méthode et tout autre paramètre, comme indiqué ci-dessous.

Math.method(parameter1, parameter2, ...);

Exemple :

Math.sin( double(Var1.getIntValue( ) ) );

Pour plus d'informations sur les méthodes mentionnées ci-dessus, consultez la documentation Java standard.

  • Le Java utilise des valeurs de précision double pour la plupart des méthodes de mathématique et les scripts Vijeo-Designer peuvent fonctionner également en précision double. Cependant, les variables Vijeo-Designer ne prennent en charge que la précision simple.
    Lors de l'écriture vers une variable flottante Vijeo-Designer, en coulisse, le script modifie le type de la valeur en un entier, qui peut causer la perte de données et de précision.

double d = Math.sin(90);

Float01.write( d );

// d est modifié

// Float01.write( float(d) );
