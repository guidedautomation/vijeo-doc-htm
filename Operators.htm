Utilisez l'opérateur Ajouter (+) pour effectuer une concaténation de chaînes. En d'autres termes, vous pouvez concaténer des chaînes avec le symbole Ajouter. Par exemple, "Bonjour " + " " + "Fred" donne la chaîne "Bonjour Fred".

Opérateurs de comparaison

Ces opérateurs comparent deux valeurs. L'expression est true lorsque la comparaison définie est vraie. L'expression est false lorsque la comparaison définie est fausse.

Opérateur

Expression

Description

>

X > Y

Supérieur à.

Retourne true lorsque X est supérieur à Y.

>=

X >= Y

Supérieur ou égal à.

Retourne true lorsque X est supérieur ou égal à Y.

<

X < Y

Inférieur à.

Retourne true lorsque X est inférieur à Y.

<=

X <= Y

Inférieur ou égal à.

Retourne true lorsque X est inférieur ou égal à Y.

==

X == Y

Egal à.

Retourne true lorsque X est égal à Y.

!=

X != Y

Différent de.

Retourne true lorsque X est différent de Y.

Opérateurs conditionnels et logiques

Opérateur

Expression

Description

&&

X && Y

AND conditionnel.

Retourne true lorsque X et Y sont vrais.

Retourne false lorsque X ou Y est faux.

Calcule Y si X est vrai.

||

X || Y

OR conditionnel.

Retourne vrai true lorsque X ou Y est vrai.

Calcule Y si X est faux.

!

!X

NOT logique.

Retourne false lorsque X est vrai.
Retourne true lorsque X est faux.

&

X & Y

AND.
Retourne true lorsque X et Y sont vrais.

Calcule toujours X et Y.

|

X | Y

OR.
Retourne true lorsque X est différent de Y (l'un est vrai, l'autre est faux).

Calcule toujours X et Y.

Opérateurs sur bit

Opérateur

Expression

Description

>>

X >> Y

Décalage à droite. Décale vers la droite les bits de X du nombre Y de positions de bit. Le signe est conservé.

<<

X << Y

Décalage à gauche. Décale vers la gauche les bits de X du nombre Y de positions de bit. Le signe est conservé.

>>>

X >>> Y

Remplissage par des zéros, décalage à droite. Décale vers la droite les bits de X du nombre Y de positions de bit et remplace les bits décalés vers la gauche par des zéros. Le signe n'est pas conservé.

&

X & Y

AND de manipulation de bits.

Retourne le résultat de l'opération AND au niveau du bit.

X

0 1 0 0

Y

1 1 0 1

Résultat

0 1 0 0

|

X | Y

OR de manipulation de bits.

Retourne le résultat de l'opération OR au niveau du bit.

X

0 1 0 0

Y

1 1 0 1

Résultat

1 1 0 1

^

X ^ Y

XOR de manipulation de bits.

Retourne le résultat de l'opération XOR au niveau du bit.

X

0 1 0 0

Y

1 1 0 1

Résultat

1 0 0 1

~

~X

Complément de manipulation de bits.

Retourne les bits de la valeur X.
