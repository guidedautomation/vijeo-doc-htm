Sys.removeProcess

Fonction

Disponible uniquement sur les machines cibles XBTGTW et iPC Series. Oblige l'arrêt du programme d'application démarré à l'aide de l'appel de la méthode Sys.createProcess.

Vous n'utiliserez normalement pas cet appel pour mettre fin à un processus. Vous le quitterez généralement depuis l'application en question.

Format

boolean Sys.removeProcess(int processHandle);

Paramètres

Paramètre

Description

int processHandle

Utilisez la valeur du descripteur retournée à partir de la méthode createProcess correspondante.

Valeur retournée

Valeur

Description

True

Le programme défini a été localisé et est contraint de se terminer.

False

Le programme n'est pas localisé ou le processus s'est terminé de façon anormale.
