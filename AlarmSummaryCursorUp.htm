cursorUp

Fonction

Permet de déplacer le curseur vers l'alarme précédente dans un affichage de résumé d'alarmes ou d'événements. Le paramètre permet de définir le type d'alarme ou d'événement à afficher.

Format

int NomRésuméAlarme.cursorUp(int Msg_State);

Paramètres

Paramètre

Description

int Msg_State

Permet de spécifier le type d'alarme ou d'événement affiché.

0: Le message précédent s'affiche.

1: Le message d'alarme actif ou non acquitté suivant s'affiche.

2: Le message d'alarme acquitté suivant s'affiche.

3: Le message d'alarme retombé à la normale suivant s'affiche.

4: Le message d'alarme actif suivant s'affiche. (ACQ obligatoire seulement.)

5: Le message d'alarme non acquitté suivant s'affiche. (ACQ obligatoire seulement.)

Valeur retournée

Les valeurs retournées suivantes indiquent les résultats de l'opération.

Valeur

Description

0

Opération réussie.

1

Curseur désactivé. Le curseur doit être activé pour utiliser cette opération.

2

Le curseur ne peut plus se déplacer vers le haut. Le curseur se trouve sur le premier message.
