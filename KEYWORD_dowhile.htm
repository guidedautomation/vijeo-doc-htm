do-while

do

{

[instructions];

}

while (expression conditionnelle);

Commentaires

Les instructions entre les mots clés do-while sont exécutées au moins une fois. Elles sont répétées aussi longtemps que l'expression conditionnelle évaluée par while est true.

Utilisez break pour quitter le bloc d'instructions.

La différence entre les boucles while et do-while est lorsque la condition est testée. Les boucles while vérifient la condition avant d'exécuter les instructions et les boucles do-while exécutent les instructions au moins une fois avant de vérifier la condition.

  • Lorsque vous utilisez des boucles (do-while, for, while), veillez à ne pas créer une boucle sans fin. Les boucles sans fin, impossibles à détecter lors de la validation, peuvent causer un blocage fatal de la machine cible étant donné que le script utilise toujours plus de mémoire.

  • Si vous utilisez des boucles do-while dans des scripts Au démarrage, A l'ouverture de l'écran, A la fermeture de l'écran, Action sur déclenchement, ou Conditionnel, et que le script ne se termine pas en cinq secondes, le script sera terminé pour que les opérations d'écran puissent continuer.

  • Si vous utilisez des boucles do-while dans des scripts sur appui, qui comprennent l'animation sur appui, les commutateurs, les scripts d'application A l'appui de l'écran, et les actions sur appui d'alarme, et que le script ne se termine pas en cinq secondes, le script sera terminé pour que les opérations Action sur appui puissent continuer.

Exemple

Le message est affiché jusqu'à ce que la valeur soit supérieure ou égale à 10.

int counter=0, value=0;

String msg;

 

do

{

++counter;

msg = "Le compteur actuel est "+counter;

value++;

}

while (value<=10);
