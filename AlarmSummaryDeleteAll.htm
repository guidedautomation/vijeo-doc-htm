deleteAll

Fonction

Cette méthode de script supprime tous les messages d'alarme affichés dans le résumé d'alarmes. La méthode peut également supprimer tous les messages d'événement affichés dans le résumé d'événements. Les paramètres de la méthode permettent de définir les types d'alarmes ou d'événements supprimés du résumé.

Si vous affectez un résumé d'alarmes à une catégorie d'alarme, la méthode de script deleteAll remet le compteur actuel à zéro. Si vous affectez un résumé d'alarmes à un groupe d'alarmes, la méthode de script deleteAll ne remet pas le compteur actuel à zéro.

On ne devra pas confondre la méthode deleteAll avec la méthode clearAll. La méthode clearAll supprime les messages du résumé et de la base de données de messages. Une autre différence entre la méthode deleteAll et la méthode clearAll est que les messages supprimés par la méthode deleteAll peuvent réapparaître dans le résumé si l'état du message change (p. ex., Haut à HautHaut, Haut à Normal, et ainsi de suite). Les messages supprimés par la méthode clearAll n'apparaîtront plus dans le résumé à moins que le message devienne actif.

  • Lorsque vous l'utilisez avec un groupe d'alarmes de diagnostic, cette méthode de script ne touche pas les alarmes dans le tampon de diagnostic de l'automate. Cette méthode de script n'interagit qu'avec l'affichage du résumé d'alarmes. Vous pouvez utilisez cette méthode sur tous les messages d'alarme dans les affichages d'alarme historiques et de journal, et sur les messages d'alarme acquittés dans les affichages d'alarme actifs.

Format

int NomRésuméAlarme.deleteAll(int Msg_State, int Disp_State);

Paramètres

Paramètre

Description

int Msg_State

Spécifie les types d'alarmes ou d'événements supprimés dans cette opération.

0: Permet de supprimer tous les messages d'alarme ou d'événement, quel que soit l'état dans lequel ils se trouvent.

1: Permet de supprimer tous les messages d'alarme d'état Actif ou Non acquitté.

2: Permet de supprimer tous les messages d'alarme d'état Retombé à la normale.

3: Permet de supprimer tous les messages d'alarme d'état Acquitté.

4: Permet de supprimer tous les messages d'alarme d'état Actif.

5: Permet de supprimer tous les messages d'alarme d'état Non acquitté.

int Disp_State

Spécifie les types d'alarmes supprimées par cette opération.

0: Permet de supprimer tous les messages d'alarme de l'affichage du résumé d'alarme.

1: Permet de supprimer uniquement les messages d'alarme visibles de l'affichage du résumé d'alarme. En d'autres termes, les alarmes de la page courante.

Valeur retournée

Retourne le nombre d'alarmes supprimées durant cette opération, exprimé par un nombre entier.

Voir également

clear, clearAll, delete

  • Si vous ouvrez un projet créé sous la version 4.1 ou sous une version antérieure, toute utilisation de clearAll dans un script sera remplacée par deleteAll.
