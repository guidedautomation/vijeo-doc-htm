18.3.3.2 Utilisation de variables de structure dans des scripts

Une structure contient plusieurs variables et chacune d'elles peut constituer un type de données différent. Un tableau est similaire à une structure en cela qu'il contient de nombreuses variables, à la différence que le type de données de chaque élément du tableau est identique.

Si vous souhaitez utiliser une variable de structure Vijeo-Designer, attribuez un nom à la variable de structure et à l'élément.
