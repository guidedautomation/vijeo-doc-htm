18.9.1.2 Méthodes de résumé d'alarmes et d'événements

Utilisez les méthodes suivantes pour interagir avec les objets de résumé d'alarmes et d'événements.

Méthode

Description

ack

Acquitte l'alarme sélectionnée dans un résumé d'alarmes ou d'événements.

ackAll

Acquitte toutes les alarmes d'état Actif dans un résumé d'alarmes ou d'événements.

Un résumé d'alarme affiche les alarmes appartenant à un groupe, pour que cette commande fonctionne de la même manière que la méthode ack du groupe d'alarmes.

clear

Supprime l'alarme sélectionnée dans le résumé d'alarmes ou d'événements du système.

clearAll

Permet d'effacer toutes les alarmes dans la base de données d'alarmes qui appartiennent au groupe d'alarmes du résumé d'alarmes. Cette méthode permet également d'effacer tous les événements dans la base de données d'événements qui appartiennent au groupe d'événements du résumé d'événements.

cursorUp

Permet de déplacer le curseur vers la ligne précédente dans un résumé d'alarmes ou d'événements.

cursorDown

Permet de déplacer le curseur vers la ligne suivante dans un résumé d'alarmes ou d'événements.

cursorOn

Permet d'afficher le curseur vers la première alarme ou le premier événement dans l'affichage du résumé.

cursorOff

Permet de supprimer le curseur du résumé d'alarmes ou d'événements.

delete

Supprime l'alarme ou l'événement sélectionné de la base de données.

deleteAll

Permet d'effacer toutes les alarmes dans le résumé d'alarmes, ou tous les événements dans le résumé d'événements.

jumpTo

Permet de déplacer le curseur vers la ligne du résumé d'alarmes ou d'événements définie dans les paramètres.

moveTop

Permet de déplacer le curseur vers la première alarme du résumé d'alarmes ou d'événements.

moveBottom

Permet de déplacer le curseur vers la dernière alarme du résumé d'alarmes ou d'événements.

pageUp

Permet de déplacer le curseur vers la page précédente dans un résumé d'alarmes ou d'événements.

pageDown

Permet de déplacer le curseur vers la page suivante dans un résumé d'alarmes ou d'événements.

showSubDisplay

Permet d'exécuter l'action sur appui de la variable sélectionnée dans la ligne du résumé d'alarmes ou d'événements.
