getSum

Fonction

Chaque groupe d'alarmes et catégorie d'alarme compte le nombre d'alarmes déclenchées. Cette opération retourne ce nombre.

Format

int NomGroupeAlarme.getSum();

int NomCatégorieAlarme.getSum();

Valeur retournée

Retourne un entier correspondant au compte historique de toutes les alarmes activées dans le groupe ou la catégorie d'alarmes.

Voir également

ackAll, clearSum, getCurrent, addMessage
