setLabel

Fonction

Permet au script de modifier l'étiquette de la recette pendant le runtime à la valeur du paramètre.

  • Change le numéro de groupe de recettes ou le numéro de recette met à jour l'étiquette de la recette. Toutefois, changer l'étiquette de la recette ne change pas la recette correspondante, donc le numéro de groupe de recettes et le numéro de recette ne changent pas.

  • L'étiquette de la recette est mise à jour lorsque la langue active est modifiée dans runtime.

  • Toute modification à l'étiquette de la recette doit être enregistrée avec une opération d'enregistrement sinon elle sera perdue lorsqu'une autre recette est chargée ou lorsque runtime est redémarré.

Format

boolean RecipeControl.setLabel(String value);

boolean RecipeControl.setLabel(String value, int LanguageID);

Paramètre

Paramètre

Description

String value

Spécifie la valeur de type chaîne à écrire dans l'étiquette de la recette.

  • Si plus d'une langue existe, la valeur de chaîne sera définie sur l'étiquette de recette pour la langue actuelle.

Paramètre

Description

String value

Spécifie la valeur de type chaîne à écrire dans l'étiquette de la recette.

int LanguageID

Spécifie l'ID de langue.

Valeur retournée

Retourne Vrai si l'étiquette de la recette a été modifiée avec succès.

Retourne Faux si l'étiquette de la recette n'a pas été modifiée en raison d'une erreur de saisie des données.

  • L'éditeur peut valider le type de données et la syntaxe mais pas la valeur. La saisie des données invalide est traitée pendant le runtime.
