Sys.setDate

Fonction

Modifie la date sur la machine cible.

Vous pouvez également modifier la date à partir du menu Configuration, mais l'utilisation de scripts permet de le faire tant que Vijeo-Designer Runtime est en cours d'exécution.

Format

boolean Sys.setDate(int year4, int month, int day);

Paramètres

Paramètre

Description

int year4

Le premier paramètre définit une année à quatre chiffres. Les valeurs acceptables sont comprises entre 1980 et 2037.

int month

Le deuxième paramètre définit un mois à un ou deux chiffres. Les valeurs acceptables sont comprises entre 1 et 12.

int day

Le troisième paramètre définit un jour à un ou deux chiffres. Les valeurs acceptables sont comprises entre 1 et 31.

Utilisez  –1 pour n'importe quel paramètre afin de l'ignorer.

Exemple :

Sys.setDate(2003, 12, -1);

// définir l'année sur 2003, le mois sur 12, et ne pas changer le jour

  • Pour les paramètres du mois et du jour, n'utilisez qu'un seul chiffre pour les numéros compris entre 1 et 9. L'utilisation de deux chiffres, comme «08» ou «09» peut produire des résultats inattendus.

Valeur retournée

Valeur

Description

false

Impossible de régler la date sur la machine cible. Indique généralement des paramètres incorrects, mais peut également signaler une erreur de communication.

true

Date correctement réglée sur la machine cible.
