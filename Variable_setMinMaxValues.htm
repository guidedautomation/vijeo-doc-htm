setMinMaxValues

Fonction

Modifie les valeurs minimum et maximum de la plage d'entrée d'une variable, limitant ainsi la plage de valeurs que vous pouvez définir pour la variable depuis la machine cible.

Active la fonction Plage d'entrée puis configure les valeurs d'entrée minimum et maximum.

Pour plus d'informations sur la plage d'entrée, consultez la section 6.2.3 Propriétés de la variable.

Format

void NomVariableEntier.setMinMaxValues(int min, int max);

void NomVariableREAL.setMinMaxValues(float min, float max);

Paramètre

Paramètre

Description

int min

Définit la valeur minimum de la plage d'entrée.

int max

Définit la valeur maximum de la plage d'entrée.

Paramètre

Description

float min

Définit la valeur minimum de la plage d'entrée.

float max

Définit la valeur maximum de la plage d'entrée.
