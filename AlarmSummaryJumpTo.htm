jumpTo

Fonction

Dans le résumé d'alarmes ou d'événements, permet de déplacer le curseur vers la ligne définie par le paramètre.

Format

void NomRésuméAlarme.jumpTo(int row_number);

Paramètres

Paramètre

Description

int row_number

Détermine la ligne du résumé d'alarmes ou d'événements vers laquelle déplacer le curseur à l'aide du paramètre de type entier (1 à 9999).

Valeur retournée

Aucun
