18.9.1.7 Méthodes DiagBuffer

Utilisez les méthodes suivantes afin d'agir sur les messages de statut stockés dans les fichiers .sto associés à l'alarme de diagnostic actuellement sélectionnée. Pour plus d'informations sur les alarmes de diagnostic, reportez-vous à la section 16.3.2, Groupes d'alarmes de diagnostic.

Méthode

Description

decodeStatusWords

Retourne les messages de statut des fichiers .sto associés à l'alarme sélectionnée.

updateStatusWords

Récupère les valeurs de mot de statut actuelles associées à l'alarme sélectionnée.

getDfbName

Retourne le nom du fichier .sto associé à l'alarme sélectionnée.

getStatusWords

Retourne un tableau entier qui stocke les mots de statut associés à l'alarme sélectionnée.
