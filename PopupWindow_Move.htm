move

Fonction

Déplace et redimensionne une fenêtre popup.

Format

boolean PopupWindowName.move(int X, int Y, int width, int height);

Le coin supérieur gauche de l'écran correspond au point d'origine (coordonnées 0,0). Les valeurs X positives déplacent la fenêtre vers la droite, les valeurs X négatives déplacent la fenêtre vers la gauche. Les valeurs Y positives déplacent la fenêtre vers le bas, les valeurs Y négatives déplacent la fenêtre vers le haut.

Paramètres

Utilisez les paramètres suivants lorsque vous effectuez cette opération. Tous les paramètres doivent être définis, même si vous vous contentez de déplacer ou redimensionner la fenêtre popup.

Paramètres

Description

int X

Définit les coordonnées de la fenêtre popup par rapport au côté gauche de l'écran graphique.

La machine cible prend en charge le déplacement par intervalles de 8 pixels. Si vous définissez un déplacement de 7 pixels, aucun déplacement n'a lieu.

int Y

Définit les coordonnées de la fenêtre popup par rapport au côté supérieur de l'écran graphique.

int width

Définit la largeur de la fenêtre.

La machine cible prend en charge les tailles de fenêtre par intervalles de 8 pixels.

int height

Définit la hauteur de la fenêtre.

La hauteur définie ici correspond à la hauteur de l'écran de la fenêtre popup sans la barre de titre.

Valeur retournée

True lorsque l'opération de déplacement s'est correctement déroulée.
