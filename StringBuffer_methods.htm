18.9.2.3 Méthodes StringBuffer

Vijeo-Designer prend en charge ces méthodes de la classe StringBuffer java.lang standard.

Pour utiliser ces méthodes, créez une variable StringBuffer, puis appelez la variable et la méthode ainsi que tout autre paramètre.

StringBuffer stringBufferVar1;  // déclarer la variable StringBuffer

char rv;

rv = stringBufferVar1.charAt(5); // méthode de variable StringBuffer

                                 // retourne un caractère dans la

                                 // position définie

Pour utiliser les méthodes StringBuffer avec une variable de type chaîne de la liste des variables Vijeo-Designer, obtenez la variable Vijeo-Designer en question et créez un nouvel objet StringBuffer.

StringBuffer s = new StringBuffer(VariableChaîne.getStringValue());
