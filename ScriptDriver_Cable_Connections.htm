18.8.4 Connexions de câble

Connectez l'équipement série au port COM de la machine cible comme défini dans la boîte de dialogue Configuration du pilote.

Vous pouvez connecter l'équipement série à l'aide d'une connexion RS-232C, RS-422 (2 fils), ou RS-422 (4 fils), selon les signaux utilisés par votre équipement série. Reportez-vous à l'annexe 5.4.1, Connexions de câble du pilote de script pour obtenir une description des connexions de câble des pilotes de script.
