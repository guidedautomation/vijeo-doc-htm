18.8.3 Configuration du pilote de script

Dans l'application Vijeo-Designer, ajoutez le pilote de script dans le nœud Gestionnaire E/S.

  1. Cliquez avec le bouton droit de la souris sur le nœud Gestionnaire E/S, puis sélectionnez Nouveau Pilote.

  1. Sélectionnez le fabricant (Pilotes génériques) et le pilote de script, puis cliquez sur OK.

  2. Cliquez avec le bouton droit de la souris sur le nœud du pilote que vous venez d'ajouter, puis sélectionnez Configuration afin d'afficher la boîte de dialogue Configuration du pilote.

  1. Dans la boîte de dialogue Configuration du pilote, définissez les paramètres de communication du pilote.

Assurez-vous que ces paramètres correspondent aux paramètres de communication série de l'équipement.

Champ

Description

Port COM

Sur la machine cible, sélectionnez le port COM dédié aux communications série.

Interface série

Sélectionnez la connexion série utilisée sur la machine cible : RS-232C, RS-422 (2 fils) ou RS-422 (4 fils).

Pour plus d'informations sur la connexion de l'équipement série, reportez-vous à la section 18.8.4, Connexions de câble.

Contrôle de flux

Aucun

Vitesse de transmission

Définit la vitesse de communication, à savoir le débit en bauds de l'équipement.

Le pilote de script prend en charge des vitesses comprises entre 2 400 et 115 200 bit/s.

Bit de parité

Définit le bit de parité utilisé pour valider les paquets de communication : Pair, Impair ou Aucun.

Bit d'arrêt

Définit le bit de d'arrêt (1 ou 2) des paquets de communication.

Longueur des données

Définit la longueur des données (7 ou 8 bits) des paquets de communication.

  1. Cliquez sur OK pour achever la configuration du pilote de script.

  2. Choisissez un nom approprié pour le pilote.
    Ce nom est utilisé dans des scripts lorsque nous envoyons/recevons des communications et les méthodes write et read .
