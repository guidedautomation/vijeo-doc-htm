switch-case-default

switch (expression)

{

case literal01 :

[les instructions sont exécutées si l'expression est égale à la valeur1];

[break];

case literal02 :

[les instructions sont exécutées si l'expression est égale à la valeur2. ];

[default] :

[les instructions sont exécutées lorsqu'aucune des valeurs case ne correspond à l'expression];

}

Les crochets [ ] indiquent des éléments optionnels. Les mots clés sont en gras.

L'expression switch doit correspondre à int ou char. Utilisez des valeurs littérales pour identifier les instructions case.

switch évalue une expression, puis passe à l'instruction case qui correspond à la valeur de l'expression.

Si la valeur évaluée n'est pas représentée dans l'une des valeurs case, le script passe aux instructions default si elles existent ou quitte le bloc switch dans le cas contraire.

Les instructions case sont exécutées les unes après les autres jusqu'à ce que le mot clé break ou la fin du bloc switch soit atteint. Dans les deux cas, l'exécution des instructionsswitch est interrompue.

Commentaires

Chaque instruction case utilise une valeur littérale incluant des entiers et des caractères. Notez que les caractères correspondent à un nombre.

Bien que le mot clé default soit optionnel, l'utiliser dans tous les jeux d'instructions constitue une bonne habitude de programmation.

Exemple

Dans l'exemple ci-dessous, switch évalue l'expression "option", accède alors à l'instruction case correspondante et écrit une valeur dans variable1.

char option;

switch (option)

{

case 'a':

variable1.write('b');

break;

case 'b':

variable1.write('c');

break;

case 'c':

variable1.write('d');

break;

default:

variable1.write('a');

}
