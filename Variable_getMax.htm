getMax

Fonction

Utilisez cette méthode avec des variables configurées dans une plage de saisie. Elle retourne la valeur maximale de la plage de saisie de la variable entière ou REAL. Vous pouvez vous assurer que les écritures vers la variable ne dépassent pas la valeur maximale avec cette méthode.

  • Si cette méthode est utilisée avec une variable n'ayant pas une plage de saisie définie, la méthode retournera la valeur maximale qui peut être stockée par la variable. Reportez-vous à la section 6.2.1, Types de variables pour plus d'informations sur les variables.

Format

int NomVariableEntiere.getMax()

float NomVariableREAL.getMax()

Valeur retournée

Retourne la valeur maximale de la variable, telle que définie dans la plage de saisie de la variable.
