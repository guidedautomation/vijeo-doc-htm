Les crochets [ ] indiquent des éléments optionnels. Les mots clés sont en gras.

return interrompt le script. Utilisez l'instruction return lorsque vous souhaitez arrêter l'exécution au milieu du code du script.

Commentaires

Aucun

Exemple

Dans l'exemple ci-dessous, switch évalue l'expression et exécute les instructions case correspondantes. Lorsque l'expression ne correspond à aucune de ces valeurs case, la valeur par défaut est exécutée et lorsque le mot clé return se présente, le script est interrompu.

int option=1;

 

switch (option)

{

case 1 :

variable1.write(1);

break;

case 2 :

variable2.write(2);

default:

return; // le script est interrompu

}

// instructions supplémentaires

Dans l'exemple, l'instruction case 1 sort de l'instruction switch et exécute les instructions supplémentaires; l'instruction case 2 quitte le script car elle exécute ses instructions case, puis les instructions default.
