18.9.1.5 Méthodes système

Utilisez les méthodes suivantes pour interagir avec le système de la machine cible.

Méthode

Description

beep

Annule le paramètre de buzzer et active le bip de la cible.

clearAllDataLoggingInMemory

Permet d'effacer toutes les données de journalisation de données de la SRAM ou la DRAM.

clearBiometricSwitch

Efface le contenu du bouton biométrique lors de l'exécution.

createProcess

Démarre un programme d'application sur la machine cible à partir du script.

exitRuntime

Quitte l'application runtime.

flushAllDataLogging

Déplace toutes les données de journalisation de données à partir du buffer dans un stockage permanent.

getInfoInt

Récupère les informations système demandées en tant que valeur entière.

getinfoString

Récupère les informations système demandées en tant que valeur de chaîne.

getLastFieldRequested

Récupère l'ID de champ du dernier objet affiché pour activer le mode de saisie.

getLastFieldUpdated

Récupère l'ID de champ du dernier objet affiché pour que la touche Entrée soit enfoncée sur le clavier.

isPlatform

Vérifie si le script est exécuté sur la machine cible (IHM) ou dans un navigateur (Web Gate).

removeProcess

Met fin à un programme d'application démarré avec createProcess.

connexion

Permet à l'utilisateur de se connecter à la machine cible.

logout

Ferme la session de l'utilisateur de sécurité actuel.

setDate

Modifie la date de l'horloge de la machine cible.

setTargetIPAddress

Permet de modifier les paramètres de réseau (Adresse IP cible, Masque de sous-réseau, Passerelle par défaut) lors de l'exécution.

setTime

Modifie l'heure de l'horloge de la machine cible.

validateUser

Vérifie le nom d'utilisateur et le mot de passe de l'opérateur qui approuve une action ou une opération protégée.
