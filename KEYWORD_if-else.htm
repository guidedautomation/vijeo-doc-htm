if-else

if (expression conditionnelle)

{

[les instructions sont exécutées lorsque l'expression conditionnelle est true];

}

[else]

{

[les instructions sont exécutées si l'expression conditionnelle est false];

}

Les crochets [ ] indiquent des éléments optionnels. Les mots clés sont en gras.

L'expression utilisée dans les instructions if doit retourner la valeur true ou false. Contrairement à la syntaxe du langage C, les valeurs numériques ne sont pas évaluées comme étant true ou false.

if évalue l'expression conditionnelle. Lorsque l'expression retourne la valeur true elle exécute le bloc d'instructions après if. Lorsque l'expression retourne la valeur false, elle passe aux instructions else, le cas échéant.

Commentaires

Les instructions conditionnelles if-else peuvent être intégrées ou imbriquées. Le nombre de niveaux n'est pas déterminé au préalable.

Exemple

Dans l'exemple ci-dessous, les variables locales temp et normal récupèrent les valeurs de température du système. Ces valeurs sont ensuite comparées. La variable msg se voit affecter différentes chaînes selon que la comparaison renvoie la valeur true ou la valeur false.

int temp, normal;

String msg;

temp = FurnaceTemp.getIntValue();

normal = FurnaceNorm.getIntValue();

 

if (temp > normal)    // comparaison

msg = "Température trop élevée";

else

msg = "Température normale";
