DiagBuffer.updateStatusWords

Fonction

Permet de récupérer les valeurs de mot de statut actuelles associées à l'alarme sélectionnée et de mettre à jour les nouvelles valeurs de mot de statut dans l'enregistrement de l'alarme correspondante.

Cette méthode ne peut être utilisée que dans un script dans une action périodique.

Format

boolean DiagBuffer.updateStatusWords(AlarmSummary AlarmSummaryName);

Paramètre

Paramètre

Description

AlarmSummary

AlarmSummaryName

Définissez le nom de l'objet du résumé d'alarmes qui affiche les alarmes de diagnostic.

Valeur retournée

Retourne la valeur Vrai si les valeurs de mot de statut ont été récupérées et mises à jour avec succès. Retourne Faux s'il n'y a aucune alarme, s'il n'y a aucune alarme sélectionnée ou si une autre erreur se produit.
