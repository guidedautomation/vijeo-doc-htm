18.6 Optimisation de l'utilisation des scripts

Vijeo-Designer offre plusieurs méthodes permettant d'optimiser l'utilisation des scripts.

Modification de la police

Les étapes suivantes décrivent comment modifier la police utilisée dans l'Editeur de scripts.

  1. Dans le menu Outils, sélectionnez Paramètres des polices de script.

  1. Dans la boîte de dialogue Paramètres des polices de script, définissez la police et la taille de la police pour le script, puis cliquez sur OK.

Enregistrement/Chargement de scripts dans/depuis des fichiers texte

Pour gagner du temps, il est possible d'enregistrer des scripts dans un fichier au format texte et de les charger à partir de ce fichier. Vous pouvez exécuter les opérations d'enregistrement et de chargement depuis le Navigateur ou l'Editeur de scripts.

  • Enregistrement dans un fichier texte
    Cliquez à droite sur le script à enregistrer, puis sélectionnez Enregistrer dans un fichier, nommez le fichier et cliquez sur OK.

  • Chargement dans un fichier texte
    Cliquez à droite sur le script à charger depuis un fichier texte, puis sélectionnez Charger depuis un fichier.

A l'apparition d'un message de confirmation, vous pouvez cliquer sur OK pour écraser le script existant avec le fichier texte sélectionné.

Si vous préférez insérer un script au lieu d'écraser l'intégralité du script existant, ouvrez et copiez le script dans un Editeur de texte, puis collez-le dans le script existant.
