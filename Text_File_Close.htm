TextFile.close

Fonction

Ferme le fichier texte ouvert et réinitialise la position de ligne du fichier actuel.

Format

int TextFile.close();

Exemple

Consultez les exemples dans 18.9.1.11 Méthodes de fichier.

Valeur retournée

Valeur retournée

Description

0

Fichier fermé avec succès.

1

Aucun fichier ouvert à fermer.

2

Impossible de fermer le fichier.

TextFile.open, TextFile.readLn
