18.3.3 Variables Tableau et Structure

18.3.3.1 Utilisation de variables Tableau Vijeo-Designer

Si vous souhaitez utiliser un tableau de variables Vijeo-Designer, signalez l'élément de tableau à l'aide de crochets [ ]. Sinon, vous pouvez utiliser les éléments de tableau de variables Vijeo-Designer comme n'importe quelle autre variable.

L'exemple suivant illustre comment utiliser une constante ou une variable pour identifier l'élément de tableau.

    Le script utilise Line_A_Speed pour calculer la vitesse de transmission de la ligne, puis écrit le résultat dans Average. L'index de tableau dans Average est défini par la valeur contenue dans la variable système Vijeo-Designer _Hour.
