DiagBuffer.getStatusWords

Fonction

Retourne un tableau de mots de statut associés à l'alarme sélectionnée.

Format

int[] DiagBuffer.getStatusWords(AlarmSummary AlarmSummaryName);

Paramètre

Paramètre

Description

AlarmSummary

AlarmSummaryName

Définissez le nom de l'objet du résumé d'alarmes qui affiche les alarmes de diagnostic.

Valeur retournée

Retourne un tableau entier de mots de statut si l'alarme sélectionnée dans le résumé d'alarmes est une alarme DFB et que des fichiers .sto sont disponibles. Pour la plupart des DFB, cette méthode retourne deux mots d'information de statut à 16 bits qui sont utilisés pour rechercher les chaînes de message d'erreur correspondantes à partir d'un fichier .sto. Reportez-vous à la documentation DFB pour obtenir des informations sur comment traiter ces mots de statut. Puisque l'on peut modifier les mots de statut sans que le tampon de diagnostic ne génère un nouvel enregistrement d'alarme de diagnostic, utilisez cette fonction dans un script périodique afin d'assurer que les mots de statut sont mis à jour dans l'application de façon continue.

De façon générale, vous utiliserez la méthode decodeStatusWords au lieu de déchiffrer les valeurs de statut vous-même.

Retourne Null si l'alarme n'est pas une alarme DFB, si des fichiers .sto ne sont pas disponibles, ou si aucune alarme n'est sélectionnée..
