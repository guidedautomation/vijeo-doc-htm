Sys.setTargetIPAddress

Fonction

Disponible sur les machines cibles équipées de deux réseaux LAN qui prennent en charge Ethernet. Permet de modifier l'adresse IP cible, le masque de sous-réseau et la passerelle par défaut de la machine cible lors de l'exécution.

  • Les machines cibles Windows (comme iPC et HMIG5U) ne prennent pas en charge le changement d'adresse IP à l'exécution et ne peuvent pas exécuter le script Sys.setTargetIPAddress.

Format

boolean Sys.setTargetIPAddress (String IP, String subnet, String gateway, Constant);

Paramètres

Paramètre

Description

String IP

La nouvelle adresse IP de la cible. La valeur est une chaîne en format décimal à quatre octets encadrée par des guillemets doubles.

String subnet

Le nouveau masque de sous-réseau. La valeur est une chaîne en format décimal à quatre octets encadrée par des guillemets doubles.

String gateway

La nouvelle passerelle par défaut. La valeur est une chaîne en format décimal à quatre octets encadrée par des guillemets doubles.

Constante

Spécifie Ethernet 1 ou Ethernet 2. Les entrées valides sont :

  • En blanc = Ethernet 1

  • ETHERNET_1 = Ethernet 1

  • ETHERNET_2 = Ethernet 2

Exemples (Ethernet 1) :

Les exemples suivants changent l'adresse IP cible d'origine à 10.187.230.100 et la définissent sur Ethernet 1. Le masque de sous-réseau et la passerelle par défaut, sans nouvelle valeur de chaîne entre guillemets, restent inchangés.

Sys.setTargetIPAddress("10.187.230.100", "", "");

Sys.setTargetIPAddress("10.187.230.100", "", "", ETHERNET_1);

Exemple (Ethernet 2) :

L'exemple suivant change l'adresse IP cible d'origine à 10.187.230.100 et la définit sur Ethernet 2. Le masque de sous-réseau et la passerelle par défaut, sans nouvelle valeur de chaîne entre guillemets, restent inchangés.

Sys.setTargetIPAddress("10.187.230.100", "", "", ETHERNET_2);

Valeur retournée

Retourne TRUE lorsque l'adresse IP cible est configurée correctement. La valeur est définie sur la variable _IPAddressStatus.

Retourne FALSE lorsque l'adresse IP cible n'est pas configurée (par exemple, lorsque le protocole DHCP est défini sur Activé ou qu'il n'existe aucune connexion réseau).
