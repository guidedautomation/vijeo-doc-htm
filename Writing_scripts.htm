18.3 Ecriture de scripts

Après avoir ajouté le script et configuré le déclenchement, écrivez le script dans l'Editeur de scripts.

Le script d'exemple ci-dessus contient cinq composants de base.

  • Variables de script : variables déclarées et utilisées localement dans le script, pour les opérations mathématiques et les expressions conditionnelles.

  • Variables Vijeo-Designer. L'éditeur de variables énumère les variables Vijeo-Designer. Pour utiliser une variable Vijeo-Designer dans le script, obtenez sa valeur et affectez-la à une variable de script.

  • Méthodes : terme de programmation orientée objet désignant une fonction ou un sous-programme. Les méthodes sont des opérations que vous pouvez exécuter sur un objet donné.

  • Mots clés du contrôle de flux : mots clés Java qui contrôlent le flux des opérations dans les scripts Vijeo-Designer. Certains mots clés Java permettent également d'ajouter de la logique à vos applications.

  • Opérateurs : vous pouvez utiliser des opérateurs pour effectuer différentes opérations mathématiques, conditionnelles ou d'affectation de type.

18.3.1 Etapes de base dans l'écriture des scripts

  1. Déclarez les variables de script.
    Reportez-vous à la section 18.3.2, Script et variables.

  1. Utilisez les méthodes de variable pour obtenir la valeur depuis une variable Vijeo-Designer et l'affecter à des variables de script.
    Reportez-vous à la section 18.3.2, Script et variables.

  1. Utilisez les opérateurs et les mots clés de contrôle de flux pour écrire le programme permettant de manipuler les variables de script locales.
    Reportez-vous aux sections 18.3.5, Opérateurs, et 18.3.6, Mots clés.

  2. Utilisez d'autres objets et méthodes. Au besoin, utilisez la méthode «write» de variables pour affecter les résultats d'opérations de script à des variables Vijeo-Designer.
    Reportez-vous aux sections 18.3.2, Script et variables, et 18.9.1, Méthodes Vijeo Designer.
