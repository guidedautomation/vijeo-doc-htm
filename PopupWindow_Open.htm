open

Fonction

Ouvre la fenêtre popup avec les coordonnées définies. Après avoir ouvert une fenêtre popup, utilisez show( ) pour en afficher la bordure et changePanel( ) pour afficher l'écran de la fenêtre popup.

Format

void PopupWindowName.open(int X, int Y);

Paramètres

Paramètre

Description

int X

Définit la coordonnée x du coin supérieur gauche de la fenêtre popup. La machine cible prend en charge les fenêtres popup par intervalles de 8 pixels uniquement.

int Y

Définit la coordonnée ydu coin supérieur gauche de la fenêtre contextuelle.

Valeur retournée

Aucun

Commentaires

Vous pouvez afficher jusqu'à trois fenêtres popup simultanément.
