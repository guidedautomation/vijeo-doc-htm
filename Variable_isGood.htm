isDataQualityGood

Fonction

Vous permet de vérifier l'intégrité de variable externes. Les variables associées à un équipement pourraient contenir des valeurs qui ne sont pas mises à jour à cause d'erreurs de communication.

Utilisez cette méthode pour vérifier l'intégrité des valeurs de variable, surtout si la variable est utilisée pour initier d'autres processus.

  • Avant d'utiliser les variables partagées ou externes, vous devriez vérifier l'intégrité des données.

Format

boolean NomVariable.isDataQualityGood();

Valeur retournée

Retourne la valeur true lorsque la valeur de variable est courante.

Retourne la valeur false lorsqu'une erreur de communication ou d'un autre type s'est produite. La valeur de variable n'est plus à jour.

Exemple 1 :

if (Integer01.isDataQualityGood())

{

// la qualité est bonne

value = Integer01.getIntValue();

}

else

{

value = -1;

}

 

Exemple 2 :

boolean quality = Integer01.isDataQualityGood();
