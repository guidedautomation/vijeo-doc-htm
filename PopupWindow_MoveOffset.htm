moveOffset

Fonction

Déplace une fenêtre popup par rapport à sa position actuelle.

Format

boolean PopupWindowName.moveOffset(int X, int Y);

Le coin supérieur gauche de l'écran correspond au point d'origine (coordonnées 0,0). Les valeurs X positives déplacent la fenêtre vers la droite, les valeurs X négatives déplacent la fenêtre vers la gauche. Les valeurs Y positives déplacent la fenêtre vers le bas, les valeurs Y négatives déplacent la fenêtre vers le haut.

Paramètres

Utilisez les paramètres suivants lorsque vous effectuez cette opération.

Paramètres

Description

int X

Définit l'importance du mouvement sur l'axe X. Une valeur positive déplace la fenêtre vers la droite, une valeur négative déplace la fenêtre vers la gauche.

La machine cible prend en charge le déplacement par intervalles de 8 pixels. Si vous définissez un déplacement de 7 pixels, aucun déplacement n'a lieu.

int Y

Définit l'importance du mouvement sur l'axe Y. Une valeur positive déplace la fenêtre vers le bas, une valeur négative déplace la fenêtre vers le haut.

Valeur retournée

True lorsque l'opération de déplacement s'est correctement déroulée.
