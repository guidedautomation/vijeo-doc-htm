Sys.beep

Fonction

Force la machine cible à émettre un seul bip sonore. Si vous souhaitez que la machine cible émette plusieurs bips sonores, appelez cette méthode plusieurs fois.

Utilisez le paramètre pour définir si le bip sonore est émis lors de l'appel ou si le bip sonore est émis selon le paramètre Buzzer dans les propriétés de cible (qui peut être également configuré dans le paramètre Buzzer sur appui du menu Configuration.)

Format

void Sys.beep(boolean ForceBeepOn);

Paramètre

Paramètres

Description

boolean ForceBeepOn

Lorsque le paramètre est vrai, le bip sonore est émis lorsque la méthode est appelée.

Lorsque le paramètre est faux, et que le buzzer est activé dans les propriétés de cible, le bip sonore est émis lorsque la méthode est appelée. Si le buzzer est désactivé, le bip sonore ne sera pas émis lorsque la méthode est appelée.

  • Les paramètres Buzzer du menu Configuration sont définis sur Aucun et A l'appui sur l'objet, ce qui correspondent aux propriétés de cible Désactivé et Activé.

Valeur retournée

Aucune

Commentaires

Utilisez cette méthode de script pour programmer un bip sonore. Par exemple, vous aimeriez peut-être avoir un bip sonore lorsque l'alarme est déclenchée (action sur déclenchement de l'alarme.)
