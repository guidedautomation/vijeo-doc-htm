DiagBuffer.decodeStatusWords

Fonction

Retourne un objet de type chaîne contenant les messages de statut des fichiers .sto associés à l'alarme sélectionnée. Chaque message est séparé par un caractère de nouvelle ligne.

Cette méthode facilite l'obtention des messages de statut. Cette méthode obtient les mots de statut et le nom du fichier du bloc de fonction dérivé (DFB) de l'alarme sélectionnée, et recherche les messages de statut dans le fichier .sto pour que vous puissiez extraire les messages.

Format

String DiagBuffer.decodeStatusWords(AlarmSummary AlarmSummaryName);

Paramètre

Paramètre

Description

AlarmSummary

AlarmSummaryName

Définissez le nom de l'objet du résumé d'alarmes qui affiche les alarmes de diagnostic.

Valeur retournée

Si l'alarme sélectionnée dans le résumé d'alarmes est une alarme de bloc de fonction dérivé (DFB) et que des fichiers .sto sont disponibles, retourne un objet de type chaîne avec les messages de statut associés. Chaque message de statut est séparé par un caractère de nouvelle ligne. La langue du système actuelle définit la langue des messages de statut. Puisque l'on peut modifier les messages de statut sans que le tampon de diagnostic ne génère un nouvel enregistrement d'alarme de diagnostic, utilisez cette fonction dans un script périodique afin d'assurer que les messages sont mis à jour dans l'application de façon continue.

Retourne une chaîne vide lorsque l'alarme n'est pas une alarme DFB, si des fichiers .sto ne sont pas disponibles, ou si aucune alarme n'est sélectionnée..
