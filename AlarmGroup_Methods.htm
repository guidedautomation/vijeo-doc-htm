18.9.1.1 Méthodes de groupe/catégorie d'alarme et de groupe d'événement

Méthodes pour groupe/catégorie d'alarme

Utilisez les méthodes suivantes pour interagir avec les groupes et catégories d'alarme.

Méthode

Description

ackAll

Acquitte toutes les alarmes d'état Actif du groupe ou de la catégorie d'alarmes spécifié.

clearSum

Efface le compte du nombre total d'alarmes générées dans le groupe ou la catégorie d'alarmes.

getCurrent

Retourne le nombre d'alarmes d'état Actif actuellement dans le groupe ou la catégorie d'alarmes.

getSum

Retourne le nombre total d'alarmes actives dans le groupe ou la catégorie d'alarmes.

 

  • Lorsque vous utilisez les méthodes Groupe d'alarmes/Catégorie pour les groupes d'événements et les catégories d'événements, Vijeo Designer les ignore lors de l'exécution.

Méthodes de groupe d'événements

Utilisez la méthode suivante pour interagir avec les groupes d'événements.

Méthode

Description

addMessage

Permet de générer un message d'événement.
