Sys.isPlatform

Fonction

Vérifie si le script est exécuté sur la machine cible (IHM) ou dans le navigateur Internet (Web Gate).

Cette méthode vous permet de créer des scripts pouvant être exécutés sur la machine cible uniquement ou sur le navigateur uniquement.

Format

boolean Sys.isPlatform(nomPlateforme);

  • platformName peut être HMI ou WEBGATE, qui sont des mots clés réservés dans les scripts Vijeo-Designer.

Paramètres

Paramètre

Description

platformName

platformName doit être HMI ou WEBGATE. Ce paramètre est sensible à la casse.

Valeur retournée

True lorsque la comparaison est vraie.
