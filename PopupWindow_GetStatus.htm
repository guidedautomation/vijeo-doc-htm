getStatus

Fonction

Retourne le statut d'affichage de la fenêtre popup.

Format

int NomFenêtrePopup.getStatus();

Valeur retournée

Les valeurs retournées suivantes indiquent les résultats de l'opération.

Valeur

Description

0

La fenêtre popup est fermée.

1

La fenêtre popup est ouverte, mais masquée.

2

La fenêtre popup est ouverte et affichée dans l'écran.
