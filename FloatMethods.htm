18.9.2.4 Méthodes Float

Vijeo Designer prend en charge ces méthodes de la classe JAVA Float standard.

Pour utiliser ces méthodes dans un script Vijeo Designer, nommez la catégorie et la méthode avec n'importe quel paramètre.

Exemple :

float f = Float.intBitsToFloat(num);

Méthode

Description

intBitsToFloat

Retourne la valeur flottante qui correspond à une représentation de bit donnée.

floatToIntBits

Retourne une représentation de la valeur de point flottant spécifiée selon la disposition de bit «format simple» du point flottant IEEE 754.

Pour plus d'informations sur les méthodes mentionnées ci-dessus, consultez la documentation Java standard.
