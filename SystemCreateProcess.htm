Sys.createProcess

Fonction

Disponible uniquement sur les machines cibles XBTGTW et iPC Series. Démarre un programme d'application sur la machine cible.

Format

int Sys.createProcess(String processName, String args);

Paramètres

Paramètre

Description

chaîne nomProcessus

Saisissez le chemin d'accès complet et le nom du fichier du programme d'application.

Utilisez deux barres obliques entre l'unité et les noms des répertoires.

Ex.: C:\\Winnt\\Notepad.exe

La barre oblique seule sert à signaler des caractères spéciaux, comme un onglet "\t" et un saut de ligne "\n". Il est donc nécessaire d'ajouter une deuxième barre oblique pour indiquer que vous ne faites pas référence à un caractère spécial.

chaîne args

Saisissez un argument ici lorsque vous y êtes invité par le programme. Saisissez une chaîne vide ("") en l'absence d'argument.

Valeur retournée

0 lorsque le processus se termine de façon anormale. Sinon, retourne la valeur du descripteur. Le numéro de ce descripteur est obligatoire pour mettre fin à un processus.

Voir également

removeProcess, isPlatform
