Sys.getInfoString

Fonction

La méthode système getInfoString() returne une chaîne qui représente l'information système. getInfoString() n'effectuera pas une scrutation en continu pour trouver des modifications aux données dans l'information système. Vous devez demander de nouvelles valeurs chaque fois que vous en avez besoin.

Format

String Sys.getInfoString(int param);

Paramètres

Paramètre

Description

int param

Une variable ou une valeur entière qui représente les informations demandées.

  • APPLICATION_BUILD_DATE

  • APPLICATION_BUILD_TIME

  • BUILD_NUM

  • DEFAULT_GTWY

  • DEFAULT_GTWY_1

  • DEFAULT_GTWY_2

  • DNS_ADDRESS_1

  • DNS_ADDRESS_2

  • IP_ADDRESS_1

  • IP_ADDRESS_2

  • MAC_ADDRESS_1

  • MAC_ADDRESS_2

  • PROJECT_NAME

  • SUBNET_MASK

  • SUBNET_MASK_1

  • SUBNET_MASK_2

  • TARGET_NAME

  • TARGET_TYPE

  • VERSION_NUM

Exemples (Adresse IP d'Ethernet 1) :

Sys.getInfoString(IP_ADDRESS_1)

Sys.getInfoString(IP_ADDRESS)

Examples (Adresse MAC d'Ethernet 2) :

Sys.getInfoString(MAC_ADDRESS_2)

Valeur retournée

Retourne une chaîne avec les informations système demandées.

Retourne une chaîne vide si les informations système demandées ne sont pas disponibles ou non prises en charge.

  • La valeur affichée par la méthode getInfoString() sont les informations système au démarrage. Ces informations ne seront pas mises à jour une fois que le programme est en cours d'exécution.

  • getInfoString() est disponible dans l'éditeur d'expressions pour l'affichage de chaîne, l'action, et le commutateur.

  • Lorsque vous utilisez la valeur retournée, par exemple dans un objet d'affichage de chaîne, assurez-vous que la longueur d'affichage est configurée à 100 pour afficher tous les caractères. Pour les chaînes retournées plus longues que 00 caractères, seuls les 100 premiers caractères sont affichés.

  • getInfoString() retourne toujours une chaîne vide lorsqu'elle est utilisée dans Web Gate.
