18.5 Modification de scripts

Lorsque vous écrivez des scripts, des commandes d'édition sont disponibles dans le menu contextuel.

  • Vous avez la possibilité de couper, copier, coller ou supprimer des blocs de texte mis en surbrillance.

  • Vous pouvez également annuler ou rétablir une opération si vous avez fait une erreur.

  • Utilisez la commande Rechercher/Remplacer pour effectuer une recherche dans le script.
    Cette commande ne s'applique qu'au script actif. Si vous voulez exécuter la même opération Rechercher/Remplacer sur d'autres scripts, sélectionnez les options appropriées dans la liste déroulante de la boîte de dialogue.
