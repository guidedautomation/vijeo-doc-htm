break

{

[instruction1];

break;

[instruction2];

}

Les crochets [ ] indiquent des éléments optionnels. Les mots clés sont en gras.

break permet de quitter le bloc d'instructions courant, qu'il s'agisse d'un bloc d'instructions case ou d'une boucle (for, while, do-while). Dans le script ci-dessus, break provoque un saut hors du bloc. L'instruction2 n'est donc pas exécutée.

Exemple

Dans l'exemple ci-dessous, switch évalue l'expression "choice" et passe à l'instruction case correspondante. Lorsque le mot clé break est rencontré, les instructions case ne sont pas exécutées.

int choice = 1;

String msg;

 

switch (choice)

{

case 1 :

variable1.write(0);

break;

case 2 :

variable1.write(1);

break;

}

Dans l'exemple suivant, index est incrémenté de 2 à la fin de chaque boucle. L'expression conditionnelle index!=9 ne peut jamais être fausse. S'il n'y avait pas de compteur, les instructions for aboutiraient à une boucle sans fin. Le compteur est incrémenté à chaque boucle et une fois que sa valeur est supérieure à 10, le mot clé break est rencontré et la boucle for est interrompue.

for (int index=0, counter=0; index!=9; index=index+2)

{

msg = "L'index actuel est " + index;

counter++;

if (counter > 10)

break;

}
