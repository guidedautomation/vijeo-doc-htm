setProperty

Fonction

Permet d'écrire le paramètre de propriété spécifié dans la SRAM de l'équipement. Le nouveau paramètre entre en vigueur dès que vous redémarrez l'équipement ou utilisez la méthode commitConfig.

  • Cette méthode de script ne permet pas de valider les valeurs de paramètres écrites vers l'équipement. Nous vous recommandons d'utiliser la méthode getProperty() pour s'assurer que les valeurs correctes sont écrites.

Format

boolean EquipmentName.setProperty(String param, int[] value);

Paramètre

Paramètre

Description

String param

Ce paramètre permet d'identifier la partie de l'équipement que vous configurez.

Ce paramètre doit être défini à l'aide de l'une des chaînes suivantes :

  • "Con_IPAddress"

  • "Con_SlaveEquipmentAddress"

  • "Con_UnitNumber"

  • "Con_DualRedundantIPAddress"

  • "Con_EquipmentAddress"

Utilisez "Con_IPAddress" pour définir l'adresse IP de l'équipement.

Utilisez "Con_SlaveEquipmentAddress" pour définir l'adresse esclave de l'équipement. (Modbus RTU seulement)

Utilisez "Con_UnitNumber" pour définir le numéro unitaire de l'équipement. (Modbus TCP/IP seulement)

Utilisez "Con_DualRedundantIPAddress" pour définir l'adresse IP de secours de l'équipement.

Utilisez "Con_EquipmentAddress" pour définir l'adresse de l'équipement pour l'esclave Modbus.

  • Si vous définissez l'adresse IP de l'équipement sur 1.1.1.1, l'équipement se met en mode Hors scrutation. Pour plus d'informations sur le mode Hors scrutation, reportez-vous à la section 5.3, Collecte de statistiques sur les pilotes.

int [ ] value

Un tableau entier disposant de quatre éléments utilisé pour définir l'adresse IP ou l'adresse esclave de l'équipement.

  • Lorsque l'adresse IP est 192.168.1.10, les valeurs d'adresse IP dans le tableau sont les suivantes : array[0]=192, array[1]=168, array[2]=1, array[3]=10.

  • Dans le tableau de chaîne Con_EquipmentAddress, le premier élément est l'ID de l'équipement.

  • Le paramètre Con_SlaveEquipmentAddress n'est disponible que pour les unités Modbus RTU.

 

Exemple

Référez-vous aux exemples dans 18.9.1.10 Méthodes d'équipement.

Valeur retournée

Retourne la valeur Vrai si le paramètre existe et que la valeur peut être écrite dans la SRAM.

Retourne la valeur Faux si le paramètre n'existe pas ou que la valeur ne peut pas être écrite dans la SRAM.
