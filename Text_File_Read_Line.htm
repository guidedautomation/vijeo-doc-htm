TextFile.readLn

Fonction

Lit une ligne de texte dans le fichier texte à la position de ligne du fichier actuel et les incréments de la ligne du fichier actuel. Le nombre maximum de caractères lus sur chaque ligne est 700.

Format

string TextFile.readLn();

Exemple

Consultez les exemples dans 18.9.1.11 Méthodes de fichier.

Valeur retournée

Retourne une chaîne du fichier texte.

Retourne une chaîne de longueur 0 pour les lignes vides.

Retourne Nul s'il a atteint la fin du fichier.

Voir également

TextFile.open, TextFile.close
