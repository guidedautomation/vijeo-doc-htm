enableScaling

Fonction

Modifie les valeurs de mise à l'échelle d'une variable externe. DataScaling définit une légende qui indique comment les valeurs sont converties entre l'équipement et la machine cible.

Active la fonction de mise à l'échelle.

Pour plus d'informations sur la mise à l'échelle, consultez la section 6.2.3 Propriétés de la variable.

  • N'utilisez pas la méthode enableScaling avec des variables internes. Même si les données de la variable sont correctement affichées, tout changement à la valeur de variable depuis la machine cible produira des résultats indésirables.

Format

void NomVariableEntier.enableScaling(

int rawMin,

int rawMax,

int scaledMin,

int scaledMax );

 

void NomVariableREAL.enableScaling(

float rawMin,

float rawMax,

float scaledMin,

float scaledMax );

Paramètre

Paramètre

Description

int rawMin

Définit la valeur minimum de la plage de données d'un équipement. Les valeurs brutes et échelonnées définissent les paramètres de conversion des valeurs entre l'équipement et la machine cible.

int rawMax

Définit la valeur maximum de la plage de données d'un équipement. Les valeurs brutes et échelonnées définissent les paramètres de conversion des valeurs entre l'équipement et la machine cible.

int scaledMin

Définit la valeur minimum de la plage de données sur la machine cible. Utilisé en parallèle à la plage de données brutes, définit les paramètres de conversion des valeurs entre l'équipement et la machine cible.

int scaledMax

Définit la valeur maximum de la plage de données sur la machine cible. Utilisé en parallèle à la plage de données brutes, définit les paramètres de conversion des valeurs entre l'équipement et la machine cible.
