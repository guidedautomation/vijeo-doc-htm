changePanel

Fonction

Change l'écran affiché dans la fenêtre popup.

Format

boolean NomFenêtrePopup.changePanel(int panel_ID);

Paramètre

Paramètre

Description

int panel_ID

Valeur de type entier qui définit l'écran (10001 à 18999) à afficher dans la fenêtre popup.

Valeur retournée

Valeur

Description

false

Echec de la modification de l'écran.

true

Opération réussie.

Commentaires

Les fenêtres popup ne peuvent afficher que les écrans affectés au nœud de fenêtre popup correspondant.

Voir également

getStatus, previousPanel, getPanelID
