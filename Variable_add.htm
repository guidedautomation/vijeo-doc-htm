add

Fonction

Cette méthode ajoute le paramètre spécifié à la variable du type entier ou REAL, ce qui est idéal pour les opérations d'incrémentation et de décrémentation. Les valeurs positives et négatives peuvent être ajoutées pour les opérations d'addition et de soustraction.

Format

boolean NomVariableEntiere.add(int value);

boolean NomVariableREAL.add(int value);;

boolean NomVariableREAL.add(float value);

boolean NomVariableREAL.add(double value);;

Paramètre

Paramètre

Description

int value

Spécifie la valeur de type entier à ajouter à la variable.

float value

Spécifie la valeur réelle à ajouter à la variable.

double value

Spécifie la valeur réelle de double précision à ajouter à la variable.

  • Les variables de type entier n'acceptent que les valeurs int en tant que paramètre.

  • Les variables du type réel n'acceptent que les valeurs int, float, et double en tant que paramètres.

Valeur retournée

Retourne true si l'opération add s'est correctement déroulée. Retourne false si l'opération add a échoué.
