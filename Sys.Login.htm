Sys.login

Fonction

Permet à l'utilisateur de se connecter à la machine cible. Les noms d'utilisateur, les mots de passe et les niveaux de sécurité sont configurés dans le nœud Sécurité. Pour plus d'informations sur la configuration de la sécurité dans votre projet, reportez-vous à la section 4.8, Configuration de la sécurité pour les cibles et les projets.

Format

void Sys.login(String UserName, String Password);

Paramètres

Paramètre

Description

String UserName

Contient le nom de sécurité de l'utilisateur.

String Password

Contient le mot de passe de l'utilisateur.

 

Valeur retournée

Aucun

 

Commentaires

Pour confirmer si l'utilisateur s'est connecté avec succès à la cible, configurez une méthode pour surveiller la valeurs des variables système _UserName et _UserLevel. Pour plus d'informations sur les variables système, reportez-vous à la section 6.7, Variables système.
