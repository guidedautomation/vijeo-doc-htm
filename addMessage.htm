addMessage

Fonction

Cette opération permet de générer un message d'événement d'utilisateur.

Format

void EventGroupName.addMessage(String str);

Paramètre

Paramètre

Description

String str

Définit une chaîne de texte à ajouter comme message d'événement défini par l'utilisateur.

Valeur retournée

Aucun
