18.9.2.2 Méthodes String

Vijeo-Designer prend en charge ces méthodes de la classe String java.lang standard. Utilisez les méthodes String pour comparer des chaînes, retourner la longueur d'une chaîne ou une chaîne modifiée, entre autres opérations.

Pour utiliser ces méthodes, créez une variable de type String et appelez la variable et la méthode ainsi que tout autre paramètre.

String stringVar1;   // déclarer la variable de chaîne

boolean rv;

rv = stringVar1.startsWith("var"); // Appeler la méthode et la variable String

                                   // Vérifier s'il commence par "var"

Pour utiliser les méthodes String avec une variable de type chaîne depuis la liste des variables de Vijeo-Designer, obtenez la variable de type chaîne Vijeo-Designer et affectez-la à une variable de script locale.

String s = VariableChaîne.getStringValue();

 

Certaines méthodes de chaîne nécessitent que la variable de chaîne soit initialisée avant de l'utiliser.

String str = "";    // Initialiser la chaîne avant de l'utiliser

Description

Retourne le caractère à l'index spécifié de cette chaîne (la numérotation commence avec l'index 0).

retour : caractère

Retourne une valeur égale à 0 si la chaîne est égale, inférieure à 0 si la chaîne est inférieure à l'argument, supérieure à 0 si la chaîne est supérieure à l'argument (sensible à la casse).

paramètre : Chaîne

retour : entier

paramètre : Chaîne

retour : Chaîne

retour : Chaîne

paramètre3 : entier

retour : Chaîne

Retourne la valeur Vrai si l'argument est un suffixe de la chaîne. Retourne la valeur Faux dans les autres cas.

retour : boolean

equals(Object anObject)

Retourne la valeur Vrai si l'argument n'est pas nul (chaîne non vide) et si l'argument est une chaîne comportant la même séquence de caractères que cette chaîne.

paramètre : Chaîne

retour : boolean

Retourne la valeur Vrai si l'argument n'est pas nul et représente la même séquence de caractères que cette chaîne sans tenir compte de la casse.

paramètre : Chaîne

retour : boolean

getBytes( )

Convertit la chaîne en octets selon l'encodage de caractère par défaut de la plate-forme et stocke le résultat dans un tableau d'octets.

retour : tableau d'octets

getChars(

int SrcBegin,

int SrcEnd,

char[ ] dest,

int dstBegin)

Copie les caractères à partir de la chaîne vers le tableau de caractères de destination.

Le premier caractère à copier est à l'index srcBegin, le dernier caractère à l'index SrcEnd moins 1. Les caractères sont copiés dans le tableau de caractères de destination, commençant à l'index dstBegin.

paramètre1 : entier

paramètre2 : entier

paramètre3 : tableau de caractères

paramètre4 : entier

paramètre : aucun

retour : entier

indexOf(int ch)

paramètre : caractère

int fromIndex)

Retourne la position de l'index dans cette chaîne de la première occurrence du caractère spécifié, à partir de la position définie de l'index, ou - 1 si le caractère n'apparaît pas.

paramètre2 : entier

int fromIndex)

toffset définit le décalage de début de cette chaîne, ooffset définit le décalage de début de l'argument de chaîne, et len définit le nombre de caractères à comparer.
