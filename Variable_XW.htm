XW

Fonction

Opération d'écriture sur bit. Permet d'écrire dans un seul bit dans une variable de type entier.

  • Avant d'utiliser les variables externes, vous devrez vérifier l'intégrité des données (isDataQualityGood).

  • Cette méthode ne fonctionne pas avec les éléments de variable de tableau, de structure, de dossier de groupe de recettes, et de bloc.

  • Lorsque vous utilisez cette méthode de script pour écrire vers la même variable entière plusieurs fois consécutivement, il est possible que seule la dernière instruction s'exécute, ce qui produit des résultats inattendus. Si vous devez modifier la valeur de plusieurs bits sur la même variable entière externe, utilsez la méthode write ou add pour modifier la valeur de la variable à l'aide d'une seule instruction.

Format

int NomVariable.XW(int position, boolean value);

Paramètre

Paramètre

Description

int position

Définit la position du bit dans lequel vous voulez écrire. La position du bit débute à 0.

boolean value

Définit la valeur d'écriture dans le bit, à savoir true ou false.

Valeur retournée

0 lorsque l'opération d'écriture a échoué.
