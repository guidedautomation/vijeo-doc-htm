getPanelID

Fonction

Retourne le numéro d'ID de l'écran affiché dans la fenêtre popup.

Format

int NomFenêtrePopup.getPanelID();

Valeur retournée

Lorsque l'opération réussit, elle retourne le numéro d'ID de l'écran affiché dans la fenêtre popup. Lorsque l'opération échoue, elle retourne -1.

Voir également

getStatus, changePanel, previousPanel
