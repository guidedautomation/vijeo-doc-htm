Sys.flushAllDataLogging

Fonction

Déplace toutes les données de journalisation de données à partir du buffer dans un stockage permanent.

La taille du buffer est configurable. Vous pouvez configurer la taille du buffer dans la propriété Nbre de registres dans la RAM dans l'Inspecteur de propriétés Groupe de journalisation; reportez-vous à la section 13.2.2, Création d'un groupe de journalisation de données. Soyez conscients que lorsqu'il est saturé, les enregistrements des variables sont déplacés automatiquement vers un support de stockage afin que de nouveaux enregistrements puissent être ajoutés au buffer.

Vous aimeriez peut-être utiliser Sys.flushAllDataLogging lorsque :

  • vous déplacerez la carte CF/SD et que vous voulez vous assurer que toutes les données de journalisation de données se trouvent dans la carte.

  • vous voulez vous assurez que toutes les données de journalisation de données se trouvent dans le dispositif de stockage au cas d'une panne d'alimentation.

Format

void Sys.flushAllDataLogging();

Valeur retournée

Aucun
