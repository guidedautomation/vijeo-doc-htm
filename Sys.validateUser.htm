Sys.validateUser.

Fonction

Vérifie le nom d'utilisateur et le mot de passe de l'opérateur qui autorise une action ou une opération protégée nécessitant approbation avant de pouvoir l'exécuter.

Les noms d'utilisateur, les mots de passe et les niveaux de sécurité sont configurés dans le nœud Sécurité. Pour plus d'informations sur la configuration de la sécurité dans votre projet, reportez-vous à la section 4.8, Configuration de la sécurité pour les cibles et les projets.

Format

void Sys.validateUser(String UserName, String UserPassword, IntegerVariable SecurityLevel);

 

Paramètres

Paramètre

Description

UserName

Le nom d'utilisateur et le mot de passe de l'opération qui autorise l'action ou l'opération.

UserPassword

SecurityLevel

La variable entière de Vijeo Designer qui stocke le niveau de sécurité de l'opérateur.

Lors de l'exécution, les valeurs écrites dans la variable seront les suivantes :

  • -1 : Indique que la validation a échoué (par exemple, un mot de passe incorrect).

  • Supérieur à 0 : Indique le niveau de sécurité de l'utilisateur lorsque la validation réussie.
