18.7.1.3 Configuration des variables et des objets

Avant d'écrire des scripts, configurez les variables, commutateurs et affichages de données suivants.

Variables

Nom

Description

D

Variable de tableau DINT comportant 15 éléments.

Adresse

Variable DINT interne utilisée pour définir la valeur d'adresse du périphérique. Cette variable stocke une valeur comprise entre 500 et 514 uniquement.

Set

Variable DINT interne qui stocke la valeur écrite dans l'adresse du périphérique.

Read_Switch

Variable BOOL interne utilisée pour déclencher l'opération permettant de lire la valeur dans l'adresse du périphérique.

Write_Switch

Variable BOOL interne utilisée pour déclencher l'opération permettant d'écrire une nouvelle valeur dans l'adresse du périphérique.
D[15]

  1. Créez une nouvelle variable DINT et désignez-la par la lettre D. Définissez Dimension du tableau sur 15 et Source de données sur Externe.

  1. Une fois que vous avez créé le tableau de variables, vous pouvez configurer facilement les adresses de périphérique pour chaque élément du tableau. Pour ce faire, cliquez avec le bouton droit de la souris sur le nœud du tableau de variables et sélectionnez Configurer l'adresse du périphérique.

  1. Cliquez sur OK pour créer le tableau.

Address, Set
  1. Créez des variables internes pour Adresse et Activer.

  1. Pour Address (valeurs comprises entre 500 et 514 uniquement), cliquez sur l'onglet Détails des données dans la boîte de dialogue Nouvelle variable et sélectionnez Plage d'entrée. Définissez 500 comme valeur minimale et 514 comme valeur maximale.

  • La définition d'une plage d'entrée permet de limiter les valeurs pouvant être saisies à partir du clavier. Cependant, les autres méthodes de modification de la valeur d'une variable (comme les scripts, les commutateurs et les opérations d'écriture depuis l'équipement) ne sont pas bloquées.

Read_Switch, Write_Switch

Créez deux variables BOOL internes nommées Read_Switch et Write_Switch

Affichages des données et commutateurs

Une fois que vous avez défini vos variables, vous avez la possibilité de configurer les affichages de données et les commutateurs associés à ces variables, lesquelles seront lues ou écrites (par les commutateurs) et affichées (par l'affichage des données) lors du runtime.

Affichage de données

  1. Configurez les objets de l'affichage des données avec la variable définie.

  2. Pour activer les saisies de clavier, cliquez sur l'onglet Mode saisie, puis sélectionnez Activer mode saisie. La case Afficher le clavier popup est sélectionnée automatiquement. Cela active le clavier popup. Retirez la coche de la case du clavier popup d'affichage pour désactiver le clavier popup.

Pour plus d'informations sur comment configurer des objets d'affichage de données, reportez-vous au chapitre 11, Affichage et saisie de valeurs.

Au lieu d'utiliser un objet d'affichage des données, vous pouvez faire appel à une animation de la valeur pour l'affichage et la saisie de données.

Commutateurs

  1. Configurez les commutateurs avec la variable définie.

  2. Configurez l'opération sur bit activé dans l'onglet A l'appui.

Pour plus d'informations sur la façon d'utiliser des commutateurs, reportez-vous à la section 9.2, Création de commutateurs.

Vous pouvez utiliser une animation sur appui à la place d'un objet commutateur pour définir et réinitialiser des variables BOOL.
