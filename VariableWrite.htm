write

Fonction

Permet au script d'interagir avec les variables Vijeo-Designer.

  • Avant d'utiliser les variables externes, vous devrez vérifier l'intégrité des données (isDataQualityGood).

  • Lorsque vous utilisez cette méthode de script pour écrire vers la même variable externe à plusieurs reprises de façon consécutive dans un très court délai (de moins de 1 seconde), certaines opérations d'écriture peuvent être supprimées afin d'optimiser l'efficacité de la communication. L'opération d'écriture finale est toujours effectuée.

  • Si chaque opération d'écriture vers la variable externe est exigée afin que l'équipement s'exécute correctement, utilisez les opérations d'écriture Bit et Mot disponibles dans les commutateurs, les affichages de texte et de message, les courbes à barre et de compteur, les sélecteurs, les touches de fonction, l'animation sur appui, et les actions d'alarme de variable.

Format

void NomVariable.write(boolean value);

void NomVariable.write(int value);

void NomVariable.write(float value);

void NomVariable.write(String value);

void NomVariableBloc.write(boolean value, int index);

void NomVariableBloc.write(int value, int index);

void NomVariableBloc.write(float value, int index);

void NomVariableBloc.write(int[] valueArray);

void NomVariableBloc.write(float[] valueArray);

Paramètre

Paramètre

Description

boolean value

Spécifie la valeur booléenne (vrai/faux) à écrire dans la variable.

La valeur booléenne peut être une variable booléenne locale ou une expression vraie ou fausse.

Paramètre

Description

int value

Spécifie la valeur de type entier à écrire dans la variable.

La valeur peut être une variable de type entier locale, une valeur de variable Vijeo-Designer (obtenue à l'aide de getIntValue(); ) ou une expression retournant un entier.

Paramètre

Description

float value

Spécifie la valeur à virgule flottante à écrire dans la variable.

Cette valeur peut être une variable de type flottant locale, une valeur de variable REAL de Vijeo-Designer (récupérée à l'aide de getFloatValue(); ) ou une expression retournant un flottant.

Paramètre

Description

String value

Spécifie la valeur de type chaîne à écrire dans la variable.

Cette valeur peut être une variable de type chaîne locale, une valeur de variable Vijeo-Designer (obtenue à l'aide de getStringValue(); ) ou une expression retournant une chaîne, comme un caractère ou un tableau de caractères.

  • Assurez-vous que le nombre de caractères dans le paramètre de chaîne ne dépasse pas la capacité de la variable. Seuls les caractères pouvant entrer dans la variable sont stockés, et les caractères excédentaires sont supprimés.

  • Utilisez des guillemets pour identifier les chaînes de texte.
    Par exemple : StringVar.write("Celle-ci est une chaîne de texte");

  • Lorsque vous utilisez des guillemets ou des barres obliques inverses, vous devez ajouter une barre oblique inverse [ \ ] dans la chaîne de texte.
    Par exemple, pour écrire la chaîne "See Spot run" said Jane, le script est : StringVar.write("\"See Spot run\"said Jane");
    Pour écrire la chaîne \\MyComputer, le script est : StringVar.write("\\\\MyComputer");

Paramètre

Description

boolean value

Spécifie la valeur booléenne (vrai/faux) à écrire vers l'élément dans une variable de bloc.

La valeur booléenne peut être une variable booléenne locale ou une expression vraie ou fausse.

  • Ecrire les variables de bloc ne met pas à jour les valeurs dans l'équipement. Utilisez sendData( ) pour écrire des valeurs à partir de la machine cible vers l'équipement.

int index

Pour les variables de bloc, spécifiez la position d'index de l'élément vers lequel vous écrivez. Les numéros d'élément débutent à 0.

Description

  • Ecrire les variables de bloc ne met pas à jour les valeurs dans l'équipement. Utilisez sendData( ) pour écrire des valeurs à partir de la machine cible vers l'équipement.
