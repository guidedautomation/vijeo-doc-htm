sendData

Fonction

Ecrit les variables de bloc depuis la machine cible vers l'équipement.

  • Avant d'utiliser les variables externes, vous devrez vérifier l'intégrité des données (isDataQualityGood).

  • Utilisez les variables de bloc avec des équipements qui prennent en charge les codes de fonction Lire multiple et Ecrire multiple afin de réaliser une meilleure performance de lecture/d'écriture. Si votre équipement ne prend pas en charge les codes de fonction Lire multiple et Ecrire multiple, les tableaux.

Format

void NomVariableBloc.sendData();
