Sys.setTime

Fonction

Modifie l'heure sur la machine cible.

Vous pouvez également modifier l'heure à partir du menu Configuration, mais l'utilisation de scripts permet de le faire tant que Vijeo-Designer Runtime est en cours d'exécution.

Format

boolean Sys.setTime(int hours, int minutes, int seconds);

Paramètres

Paramètre

Description

int hours

Le premier paramètre définit les heures sur la base d'une horloge de 24 heures. Ce paramètre peut contenir jusqu'à deux chiffres, et les valeurs acceptables sont comprises entre 0 et 23. 0 équivaut à minuit. Si vous définissez la valeur 24, elle est automatiquement convertie en 0.

int minutes

Le deuxième paramètre définit les minutes (0-59). Il peut contenir jusqu'à deux chiffres.

int seconds

Le troisième paramètre définit les secondes (0-59). Il peut contenir jusqu'à deux chiffres.

Utilisez  –1 pour n'importe quel paramètre afin de l'ignorer.

Exemple :

Sys.setTime(12, 0, -1);

// définir les heures à 12, les minutes à 0 et ne pas changer les secondes

Valeur retournée

Valeur

Description

false

Impossible de régler l'heure sur la machine cible. Indique généralement des paramètres incorrects, mais peut également signaler une erreur de communication.

true

Heure correctement réglée sur la machine cible.
