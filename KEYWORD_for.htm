for

for (initialize; conditional_expr; expression)

{

[instructions];

}

Les crochets [ ] indiquent des éléments optionnels. Les mots clés sont en gras.

Commentaires

initialize Evaluée une fois pour initialiser les variables.

condition_expr Expression conditionnelle qui contrôle l'interruption de la boucle. Tant que cette expression conditionnelle est true, les instructions for sont répétées.

expression Évaluée après les instructions for en général pour modifier la valeur des variables de conditional_expr.

Utilisez le mot clé break pour quitter la boucle prématurément, transférant ainsi le contrôle à l'instruction située après la boucle for.

  • Lorsque vous utilisez des boucles (do-while, for, while), veillez à ne pas créer une boucle sans fin. Les boucles sans fin, impossibles à détecter lors de la validation, causent un blocage fatal de la machine cible étant donné que le script utilise toujours plus de mémoire.

  • Si vous utilisez des boucles for dans des scripts Au démarrage, A l'ouverture de l'écran, A la fermeture de l'écran, Action sur déclenchement, ou Conditionnel, et que le script ne se termine pas en cinq secondes, le script terminera pour que les opérations d'écran puissent continuer.

  • Si vous utilisez des boucles for dans des scripts sur appui, qui comprennent l'animation sur appui, les commutateurs, les scripts d'application A l'appui de l'écran, et les actions sur appui d'alarme, et que le script ne se termine pas en cinq secondes, le script sera terminé pour que les opérations Action sur appui puissent continuer.

Exemple

Le message est répété jusqu'à ce que l'expression c <= 10 soit fausse.

String msg;

 

for (int c=1; c <= 10; c++)

{

msg = "Le compteur actuel est " + c;

}
