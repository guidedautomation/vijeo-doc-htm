commitConfig

Fonction

Permet d'écrire les nouveaux paramètres vers la SRAM de l'équipement et de réinitialiser la connexion Ethernet pour mettre à jour l'équipement avec les nouveaux paramètres. Vous ne devez pas redémarrer l'équipement.

Format

void EquipmentName.commitConfig();

Exemple

Référez-vous aux exemples dans 18.9.1.10 Méthodes d'équipement.

Valeur retournée

Aucun

Voir également

getProperty, getStringProperty, openConfig, setProperty
