18.2 Types de scripts

Il existe trois types de script : Ecran, Sur appui, et Alarme. et à l'usage que vous voulez en faire.

Exécution du script :

Utilisez :

Lorsqu'un écran spécifique est ouvert

Script d'écran

Lorsque vous voulez exécuter le script uniquement quand l'écran associé est ouvert, utilisez les scripts d'écran.

Pour plus d'informations sur la façon de créer un script d'écran, reportez-vous à la section 9.4, Création d'actions.

Lorsqu'un objet est touché

Script d'appui

Si vous voulez exécuter le script lorsqu'un objet est touché, ou lorsqu'une touche de fonction est touchée, utilisez un script d'appui.

Il existe trois manières de configurer des scripts d'appui : en ajoutant une animation sur appui à un objet et une opération Script, en dessinant un commutateur et en ajoutant une opération Script, ou en ajoutant une opération Script à une touche de fonction.

  • Lorsque vous utilisez des scripts sur appui, puisque l'éditeur de scripts est une boîte de dialogue, pour ajouter des espaces [Tab], appuyez sur [Ctrl]+[Tab].

Lors du déclenchement d'une alarme

Scripts d'alarme

Lorsque vous voulez exécuter le script quand une alarme se déclenche, configurez un script de déclenchement d'alarme dans la variable associée.

  • Pour ajouter des espaces [Tab] lorsque vous utilisez des scripts de déclenchement d'alarme, appuyez sur [Ctrl]+[Tab] puisque l'éditeur de scripts est une boîte de dialogue.
