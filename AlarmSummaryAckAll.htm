ackAll

Fonction

Acquitte toutes les alarmes ou tous les événements actifs ou uniquement les alarmes ou les événements visibles dans l'affichage du résumé.

Format

int NomduRésumé.ackAll(int Disp_State)

Paramètres

Paramètre

Description

int Disp_State

Spécifie la plage des alarmes ou événements à acquitter.

0: Permet d'acquitter toutes les alarmes ou tous les événements actifs dans l'affichage du résumé.

1: Permet d'acquitter toutes les alarmes ou tous les événements actifs visibles dans l'affichage du résumé.

Valeur retournée

Retourne le nombre d'alarmes ou d'événements acquittés dans l'opération.
