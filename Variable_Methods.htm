18.9.1.6 Méthodes de variable

Utilisez les méthodes suivantes pour interagir avec les variables de la liste des variables.

Méthode

Description

ack

Change l'état d'alarme de la variable de Actif à Acquitté.

add

Ajoute le paramètre défini à la variable. Les valeurs de paramètre peuvent être négatives pour la soustraction, ou positives pour l'addition.

enableScaling

Définit les paramètres de conversion de données pour les variables externes.

flushToStorage

Vide l'historique des variables du buffer dans le lecteur secondaire.

getIntValue

Retourne une valeur de type entier de la variable Vijeo-Designer.

getIntValues

Retourne un tableau du type entier lors de l'utilisation avec une variable du type entier de bloc.

getFloatValue

Retourne une valeur de type réel dans la variable Vijeo-Designer.

getFloatValues

Retourne un tableau réel lors de l'utilisation avec une variable REAL de bloc.

getMax

Utilisez cette méthode avec des variables configurées dans une plage de saisie. Elle retourne la valeur maximale de la plage de saisie de la variable entière ou réelle.

getMin

Utilisez cette méthode avec des variables configurées dans une plage de saisie. Elle retourne la valeur minimale de la plage de saisie de la variable entière ou réelle.

getSize

Retourne le nombre d'unités (éléments) dans la variable de bloc.

getStringValue

Retourne une valeur de type chaîne dans la variable Vijeo-Designer.

isDataQualityGood

Retourne une valeur true lorsque la valeur de la variable est fiable.

sendData

Ecrit les valeurs de variable de bloc définies vers l'équipement.

setMinMaxValues

Définit les valeurs minimales est maximales que vous pouvez saisir à partir de la machine cible.

toString

Retourne un objet de type chaîne pour le nom de la variable.

write

Ecrit la valeur du paramètre dans la variable Vijeo-Designer.

X

Retourne la valeur d'un bit distinct dans une variable de type entier.

XW

Ecrit dans un bit distinct dans une variable de type entier.
