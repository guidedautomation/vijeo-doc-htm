moveBottom

Fonction

Permet de déplacer le curseur vers le dernier message d'alarme ou d'événement de l'affichage du résumé.

Format

int NomRésuméAlarme.moveBottom(int Msg_State);

Paramètres

Paramètre

Description

int Msg_State

Permet de spécifier le type d'alarme ou d'événement vers lequel le curseur peut se déplacer.

0: Permet d'accéder au dernier message quel que soit l'état dans lequel il se trouve.

1: Permet d'accéder au dernier message d'alarme d'état Actif ou Non acquitté.

2: Permet d'accéder au dernier message d'alarme d'état Acquitté.

3: Permet d'accéder au dernier message d'alarme d'état Retombé à la normale.

4: Permet d'accéder au dernier message d'alarme d'état Actif. (ACQ obligatoire seulement.)

5: Permet d'accéder au dernier message d'alarme d'état Non acquitté. (ACQ obligatoire seulement.)

Valeur retournée

Les valeurs d'entier suivantes indiquent le résultat de l'exécution.

Valeur

Description

0

Opération réussie.

1

Curseur désactivé. Le curseur doit être activé pour utiliser cette opération.

2

Le curseur ne peut plus se déplacer vers le bas. Le curseur se trouve sur le dernier message.
