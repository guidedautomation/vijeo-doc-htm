18.9.1.11 Méthodes de fichier

Utilisez les méthodes suivantes pour lire ligne par ligne à partir d'un fichier texte (.txt). Le codage du fichier texte doit être Unicode. Pour plus d'informations sur l'ajout d'un fichier texte à votre projet, reportez-vous à la section 11.7.4, Ajout de fichiers texte dans la cible.

 

Méthode

Description

TextFile.open

Ouvre le fichier texte au moment de l'exécution.

TextFile.readLn

Lit à partir du fichier texte une ligne à la fois.

TextFile.close

Ferme le fichier texte.

 

Exemple

L'exemple suivant montre comment ouvrir, lire et fermer un fichier texte au moment de l'exécution.

String s; //Java string variable

int lineNum = 0;

TextFile.open("File1"); //File1 est l'alias du fichier texte

while(true)

{s= TextFile.readLn(); //retourne les 700 premiers caractères dans la ligne

if (s==null) break;

STRING01[lineNum++].write(s); //STRING01 est une variable de tableau de chaîne Vijeo Designer

}

TextFile.close();

 

  • Assurez-vous que la variable de tableau de chaîne (STRING01, dans l'exemple ci-dessus) contient assez d'éléments pour traiter le nombre de lignes dans le fichier texte. Utilisez la propriété NumOfBytes de la variable de tableau pour définir la longueur des éléments de tableau. Notez que le nombre maximum de caractères qu'une variable string peut contenir est 100 (p. ex., NumOfBytes=100). Si une ligne unique contient 700 caractères, les 600 derniers caractères seront tronqués. Pour plus d'informations, reportez-vous à la section 6.2.3.2, Onglet Détails des données.
