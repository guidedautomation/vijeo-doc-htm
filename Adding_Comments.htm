18.3.7 Ajout de commentaires

Pour ajouter des commentaires dans un script, débutez le commentaire par deux barres obliques "//".

// Les commentaires commencent par des barres obliques.

if (LineA_Speed.getIntValue( ) <= MAX)

Vous pouvez également ajouter des commentaires à la fin d'une ligne.

if (LineA_Speed.getIntValue( ) <= MAX)  // Commentaire à la fin de la ligne

 

Les scripts Vijeo-Designer ne prennent pas en charge le format de commentaire suivant.

/* Cette ponctuation n'introduit PAS un commentaire */
