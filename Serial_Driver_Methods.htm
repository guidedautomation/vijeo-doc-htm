18.8.2 Méthodes pour pilote de script

Utilisez les méthodes suivantes pour lire ou écrire sur le port de communication défini par le pilote de script.

Pour plus d'informations sur comment configurer le pilote de script pour ces méthodes, reportez-vous à la section 18.8.3, Configuration du pilote de script.

Méthode

Description

read

Lit les flux d'octets depuis le port série défini.

write

Envoie les flux d'octets au port série défini.
