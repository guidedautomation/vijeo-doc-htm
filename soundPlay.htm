Sound.play

Fonction :

Lit une fois le fichier son défini.

Format

void Sound.play(String alias);

Paramètre

Paramètres

Description

String alias

Saisissez l'alias du fichier son utilisé par cette méthode, comme défini dans le nœud Fichiers de données.

P. ex. : Sound.loop("MonAliasdeSon");

Valeur retournée

Aucune

Commentaires

Le fichier son doit faire partie de l'application utilisateur et être ajouté au nœud Fichiers de données dans Vijeo-Designer.

Voir également

Sound.loop, Sound.stop
