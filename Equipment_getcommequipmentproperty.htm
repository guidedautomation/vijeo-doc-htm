getProperty

Fonction

Retourne un tableau entier contenant les valeurs des propriétés de l'équipement.

Format

int[]EquipmentName.getProperty(String param);

Paramètre

Paramètre

Description

String param

Une variable de chaîne disposant des informations de configuration demandées par l'équipement.

  • Les paramètres de chaîne ci-dessous ne fonctionnent que si votre pilote de périphérique utilise les paramètres de communication correspondants. Veuillez tester votre équipement pour assurer le fonctionnement.

Con_IPAddress

Retourne un tableau entier contenant la valeur de l'adresse IP de l'équipement.

Con_DualRedundantIPAddress

Retourne un tableau entier contenant la valeur de l'adresse IP de l'équipement de sauvegarde. Ce paramètre est utile lorsque la communication est rompue entre l'équipement principal et la cible et que vous avez besoin de l'adresse IP de l'équipement de sauvegarde.

Con_EquipmentAddress

Retourne un tableau entier contenant la valeur de l'adresse de l'équipement.

Con_SlaveEquipmentAddress

Retourne un tableau entier contenant la valeur de l'adresse esclave de l'équipement.

  • Ce paramètre n'est pris en charge que sur les unités Modbus RTU.

Con_UnitNumber

Retourne un tableau entier contenant la valeur du numéro unitaire de l'équipement.

  • Ce paramètre est pris en charge uniquement sur les équipements Modbus TCP/IP.

Con_LocalStationAddress

Retourne un tableau entier contenant la valeur de l'adresse de la borne locale.

Con_HardwareProtocol

Retourne un tableau entier contenant la valeur de l'interface série de l'équipement.

Exemple :

Valeur retournée

Retourne un tableau entier pour le paramètre d'équipement ou une valeur nulle.
