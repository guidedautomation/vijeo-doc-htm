DiagBuffer.getDfbName

Retourne le nom du bloc de fonction dérivé (DFB) associé à l'alarme sélectionnée.

Fonction

Retourne un objet de chaîne peuplé par le nom de l'instance DFB.

Format

String DiagBuffer.getDfbName(AlarmSummary AlarmSummaryName);

Paramètre

Paramètre

Description

AlarmSummary

AlarmSummaryName

Définissez le nom de l'objet du résumé d'alarmes qui affiche les alarmes de diagnostic.

Valeur retournée

Si l'alarme sélectionnée dans le résumé d'alarmes est une alarme DFB et que des fichiers .sto sont disponibles, retourne un objet de type chaîne avec le nom associé à l'instance DFB.

Retourne une chaîne vide lorsque l'alarme n'est pas une alarme DFB, si des fichiers .sto ne sont pas disponibles, ou si aucune alarme n'est sélectionnée..
