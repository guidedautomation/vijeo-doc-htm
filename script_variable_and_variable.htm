18.3.2 Scripts et variables

Lorsque vous utilisez des scripts Vijeo-Designer, deux types de variables sont disponibles :

  • les variables Vijeo-Designer

  • les variables de script

Les variables créées dans Vijeo-Designer ne peuvent pas être utilisées directement dans les expressions de script. Pour utiliser une variable Vijeo-Designer dans un script, appliquez la méthode get value de la variable et affectez le résultat à une variable de script locale. Vous pouvez ensuite utiliser la variable de script dans les instructions de contrôle du script, les opérations mathématiques et autres expressions de script.

  • Lorsque vous utilisez des variables Vijeo-Designer dans des scripts, la valeur doit être affectée à une variable de script locale. La variable Vijeo-Designer et la variable de script locale sont indépendantes, et leurs valeurs peuvent être différentes. Elles peuvent donc avoir des valeurs différentes. Cela pourrait créer des problèmes, en particulier avec les variables externes, car la valeur de la variable Vijeo-Designer peut changer au cours d'un processus de script, alors que le script continue d'utiliser la valeur de la variable précédente.

Nom de la variable de script

Les conditions suivantes s'appliquent aux noms des variables de script.

  • Les variables de script sont sensibles à la casse. Par exemple : Int01 et int01 sont considérées comme deux variables différentes.

  • Il est impossible d'utiliser un chiffre comme premier caractère dans un nom de variable de script.

  • Il est impossible d'utiliser les mots clés Java et Vijeo-Designer comme noms de variable de script.

  • Il est impossible d'utiliser les noms des variables Vijeo-Designer comme noms de variable de script.

Initialisation des variables

Une fois la variable de script locale déclarée, prenez de bonnes habitudes de programmation en initialisant sa valeur.

En particulier, avant d'utiliser la variable de script dans une comparaison, affectez-lui une valeur.

Les variables de chaîne doivent être également initialisées avant de les utiliser dans certaines méthodes de chaîne Java.

L'exemple suivant illustre l'importance de l'initialisation des variables de script.

int a, b=0; // Seule b est initialisée à 0

if(a==0)          // Comparaison pas possible, a n'est pas défini

if(b==0)          // Comparer avec 0

     String str = "";    // Initialiser la chaîne avant de l'utiliser

 

Marqueur de fin d'instruction, le point virgule (;)

A la fin de chaque instruction, n'oubliez pas d'ajouter un point virgule (;). Sans ce point virgule, le script ne sera pas validé et générera des erreurs de syntaxe.

Exemples de déclarations de variables

short a;        // court

int b = 0;      // entier initialisé sur 0

long c, d=0, e; // 3 variables de type long, d initialisée à 0

char f[];       // tableau de caractères

char[] f = new char[100]; // tableau de caractères avec 100 éléments
