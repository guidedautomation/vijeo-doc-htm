new

Servez-vous du mot clé new pour utiliser des tableaux ou créer une instance de classes String et StringBufferv.

int a[] = new int[10];

String s = new String();

StringBuffer S = new StringBuffer();

Ne créez pas d'objets String ou StringBuffer, ni de tableaux primitifs Java occupant un espace mémoire volumineux.

Par exemple, si vous écrivez le script suivant, tout l'espace mémoire est occupé et cause l'interruption des opérations par Vijeo-Designer Runtime.

int a[] = new int[65532];  // Arrête Vijeo-Designer Runtime
