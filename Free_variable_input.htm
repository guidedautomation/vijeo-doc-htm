18.7.1 Définition de l'adresse du périphérique lors du runtime

18.7.1.1 Environnement d'exploitation

Le script d'exemple est configuré pour s'exécuter dans l'environnement d'exploitation défini.

Machine cible

Peut être configuré pour s'exécuter sur toutes les machines cibles.

Equipement

Schneider Electric PLC  Modbus (RTU)

(Connectez la machine cible au port de programmation de l'UC.)

Adresses de périphérique

30500 à 30514 (15 mots)

 

  • Dans les instructions suivantes, vous devez configurer le pilote de périphérique dans Vijeo-Designer. Reportez-vous au chapitre 5 Communications.

18.7.1.2 Opération Script

Vous pouvez définir l'adresse à lire ou à écrire à l'aide d'un tableau d'entiers Vijeo-Designer pour stocker des adresses de périphérique comprises entre 30500 et 30514.

Voici une description du mode de fonctionnement de l'application lors du runtime.

Valeur de lecture

  1. Définissez l'adresse du périphérique. (Dans cet exemple, nous utilisons des adresses de périphérique comprises entre 30500 et 30514.)

  2. Appuyez sur le commutateur Lecture.

  3. La valeur stockée dans l'adresse du périphérique s'affiche dans l'objet Affichage des données.

Valeur d'écriture

  1. Servez-vous d'un clavier pour modifier la valeur qui a été lue à partir de l'adresse du périphérique.

  2. Appuyez sur le commutateur Ecriture pour écrire la nouvelle valeur dans l'adresse du périphérique.
