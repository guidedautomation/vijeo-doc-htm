18.2.2 Scripts d'alarme

Les scripts d'alarme s'exécutent lorsqu'une alarme se déclenche ou lorsqu'un message d'alarme est sélectionné dans l'objet résumé d'alarme.

  • Si vous souhaitez exécuter un script lorsqu'une alarme devient active, configurez une Alarme Actions sur déclenchement.

  • Si vous souhaitez exécuter un script lorsque le message d'alarme est sélectionné dans un affichage de résumé d'alarme, configurez une Alarme Actions sur appui.

  • Si un script de déclenchement d'alarme utilise des boucles while ou for et ne se termine pas en cinq secondes, ou si un script Action sur appui utilise des boucles while ou for et ne se termine pas en deux secondes, le script sera terminé pour les opérations d'écran puissent continuer.

Ajout des scripts de déclenchement d'alarme

Configurez les scripts de déclenchement d'alarme à partir des propriétés des alarmes d'une variable. Vous pouvez configurer une action sur appui, telle que décrite dans les étapes suivantes, ou une action sur déclenchement. Pour plus d'informations sur comment configurer une action sur déclenchement, reportez-vous à la section 16.3.4.1, Actions sur déclenchement.

  1. Dans l'éditeur de variables, cliquez deux fois sur le numéro de ligne d'une variable pour afficher la boîte de dialogue Propriétés de la variable.

  1. Dans l'onglet Alarmes, cochez la case Alarme puis cliquez au choix sur le bouton Action sur appui ou Action sur déclenchement.

  2. Cliquez sur Ajouter, puis sélectionnez Script en tant que Fonction.

  1. Cliquez deux fois sur le champ Condition puis définissez la condition qui doit être vraie pour que l'opération soit exécutée.

Propriété

Description

Toujours

Action sur appui. Le script s'exécute lorsque le message d'alarme est sélectionné dans le résumé d'alarme.

Action sur déclenchement. Le script s'exécute lorsque l'alarme se déclenche.

Conditionnel

Condition d'alarme :

Disponible lors de la configuration des variables Entier ou Flottant. Configurez ce paramètre pour exécuter le script lorsqu'une alarme atteint la limite d'alarme.

 

Par exemple, lorsque le Type d'alarme est Limite, vous pouvez exécuter le script pour une limite d'alarme (Très bas, Bas, Haut ou Très haut) en sélectionnant la limite dans le champ Condition d'alarme.

AND, OR

Lorsque les cases à cocher Condition d'alarme et Si sont cochées, vous pouvez définir si le script est exécuté lorsque les deux instructions sont vraies (AND) ou lorsque l'une ou l'autre des instructions est vraie (OR).

Si

Cochez la case Si puis définissez une variable ou une expression. Le script s'exécute lorsque la valeur de la variable ou de l'expression est différente de zéro.

En d'autres termes, le script ne s'exécute pas lorsque la valeur est égale à zéro.

Pour plus d'informations sur l'écriture d'expressions, reportez-vous à la section 6.14, Affectation des variables à l'aide des expressions.

  1. Cliquez deux fois sur la cellule Opération pour accéder à la boîte de dialogue Script et écrire le script.

Pour obtenir des instructions sur l'écriture de scripts, reportez-vous à la section 18.3, Ecriture de scripts.

  • Pour ajouter des espaces [Tab] lorsque vous utilisez des scripts de déclenchement d'alarme, appuyez sur [Ctrl]+[Tab] puisque l'éditeur de scripts est une boîte de dialogue.
